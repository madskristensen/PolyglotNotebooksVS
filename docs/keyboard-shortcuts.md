# Keyboard Shortcuts

A complete reference of keyboard shortcuts available in the Polyglot Notebooks editor. These shortcuts are scoped to the notebook editor and won't interfere with other Visual Studio shortcuts.

---

## Execution

| Action | Shortcut |
|--------|----------|
| Run cell and advance to next | **Shift+Enter** |
| Run all cells | **Ctrl+Shift+Enter** |
| Interrupt execution | **Ctrl+.** (Ctrl+Period) |

## Cell Management

| Action | Shortcut |
|--------|----------|
| Insert code cell above | **Ctrl+Shift+A** |
| Insert code cell below | **Ctrl+Shift+B** |
| Insert Markdown cell above | **Ctrl+Shift+Alt+A** |
| Insert Markdown cell below | **Ctrl+Shift+Alt+B** |
| Move cell up | **Ctrl+Alt+↑** |
| Move cell down | **Ctrl+Alt+↓** |
| Delete cell | **Ctrl+Shift+Delete** |

## Editing

| Action | Shortcut |
|--------|----------|
| Toggle Markdown edit/preview | **F2** |
| Change cell language | **Ctrl+Shift+L** |
| Clear cell output | **Ctrl+Shift+Backspace** |

## Standard Visual Studio Shortcuts

These standard VS shortcuts also work inside notebook code cells:

| Action | Shortcut |
|--------|----------|
| IntelliSense / Completions | **Ctrl+Space** |
| Signature Help | **Ctrl+Shift+Space** |
| Undo | **Ctrl+Z** |
| Redo | **Ctrl+Y** |
| Copy | **Ctrl+C** |
| Cut | **Ctrl+X** |
| Paste | **Ctrl+V** |
| Select All (within cell) | **Ctrl+A** |
| Find | **Ctrl+F** |

## Tips

- **Shift+Enter is your most-used shortcut.** It runs the current cell and moves to the next one, creating a smooth top-to-bottom execution flow.
- **Ctrl+Shift+L** opens the kernel language picker dropdown, so you can switch a cell's language without using the mouse.
- **F2 on Markdown cells** toggles between the rendered preview and the raw source editor.

## Customizing Shortcuts

You can change these shortcuts through Visual Studio's keyboard settings:

1. Go to **Tools → Options → Environment → Keyboard**.
2. Search for **"Polyglot Notebooks"** to find all notebook commands.
3. Assign or change the key bindings as you prefer.

Each command is listed with the prefix **Polyglot Notebooks:** followed by the action name.
