## Learnings (2026-03-27)

### Architecture Decisions
- **Reference implementation**: BookmarkStudio at C:\Users\madsk\GitHub\BookmarkStudio is the canonical pattern source for all VSIX projects for Mads Kristensen.
- **csproj format**: Legacy SDK (ToolsVersion=15.0), NOT the new SDK-style csproj. ProjectTypeGuids must include {82b43b9b...} (VSIX) and {FAE04EC0...} (C#).
- **TargetFrameworkVersion**: v4.8 (net48) for all VSIX and test projects.
- **NuGet packages**: Community.VisualStudio.Toolkit.17 v17.0.549, Microsoft.VSSDK.BuildTools v17.14.2120, System.Text.Json v8.0.5.
- **Test SDK**: MSTest.Sdk/4.0.1 with UseVSTest=true and UseWPF=true. New SDK-style (not legacy).
- **Solution format**: .slnx (XML), not legacy .sln. Includes AnyCPU/arm64/x86 platform configurations.
- **Manifest target**: VS [17.0, 18.0) both amd64 and arm64 InstallationTarget entries.
- **source.extension.cs**: Auto-generated by VSIX Synchronizer 1.0.45. Contains Vsix sealed partial class with Id, Name, Description, Language, Version, Author, Tags, IsPreview consts.
- **Package entry point**: Inherits ToolkitPackage (not AsyncPackage). Uses global usings for Community.VisualStudio.Toolkit, Microsoft.VisualStudio.Shell. InitializeAsync uses JoinableTaskFactory pattern.
- **CI**: GitHub Actions build.yaml — checkout@v5, setup-msbuild@v3, cache@v5, vsix-version-stamp@v2, msbuild /v:m -restore /p:OutDir=\_built, dotnet vstest, dorny/test-reporter@v2.6.0, upload-artifact@v7, openvsixpublish@v1, VsixPublisherAction@1.0.
- **Marketplace publish trigger**: [release] in commit message or workflow_dispatch.
- **DoNotParallelize**: All test assemblies use [assembly: DoNotParallelize] in AssemblyInfo.cs.
